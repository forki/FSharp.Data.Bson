BsonProvider

Typed representation of a BSON document.

Static parameters

Static parametersDescription
Path
Signature: string

Location of a BSON sample file.
InferLimit
Signature: int (optional, default = 100)

Number of documents to use for inference. Defaults to 100. If this is set as zero, then all documents are used.
ResolutionFolder
Signature: string (optional, default = "")

Directory used for resolving relative file references (at design time and in hosted execution).
